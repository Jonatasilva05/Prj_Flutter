# Prj_Flutter
Primeiro contato e desenvolvimento com Flutter

---

## Primeiro passo para usar o Flutter é saber se o git esta instalado

<br>

- No terminal digite:

```bash
    git --version
```

se ele estiver instalado retornará uma versão do git. Caso não esteja instalado procure a documentação para saber como instalar

<br>

Após isso abra o terminal e digite

```bash
    flutter doctor
```

se retornar erro ou demorar muito a dar um retorno feche e abra o git bash.

No terminal do GIT BASH digite

```bash
    curl --tlsv1.2 https://flutter-io.cn
```

e deixe executar apos isso abra o terminal do computador novamente e digite o primeiro passo se retornar sem erro funcionou
